#### 行

实行每工作日限行2个号（仅燃油私家车）的限行政策（周一限行尾号1和6，以此类推）。高新区交通比较拥堵，软件公司较多的软件新城和环普软件园暂无地铁经过。通勤可以选择公共交通、自驾（摩托车、自行车、汽车）、拼车、打车（高峰时期难打车）、错峰出行（有的公司上班是10点开始）等方式。
