<details>
  <summary>国企、私企、外企怎么选</summary>
  
我曾在国企、私企呆过，也曾创业过，目前在一家外企。从我个人有限的认知看：
- 国企等级森严，对学历和背景要求高，岗位比较稳定，但时代变化太快。目前很多国企会有子公司化的趋势，这就意味着并不存在永久的铁饭碗，失业下岗也是可能的。且国企都是围绕着大领导为核心，节奏可能时而很闲时而很忙。可能因为岗位比较稳定的原因，同事之前的关系一般走的比较近。
- 私企的情况比较混乱，大的私企如华为之类，有国企化的氛围。小的私企如一些创业公司，老板可能随时跑路，套路员工的情况时有发生。一些创业公司更是富了老板，穷了员工。很多私企并不按照劳动法缴纳足额的五险一金。
- 外企是很遵守劳动法的，所以福利一般比较好，假期和五险一金，甚至商业保险，节假日福利都做的不错。工作的强度也不大，一般也不会随意辞退员工，大部分能做到`work life balance`。如果你对收入没有很高的要求，外企是个不错的选择。

如何选择？如果你对收入要求很高，那可以基于收入去选择，只能能在承受加班的极限范围内选择收入最高的工作就行。如果学历和能力都还可以，选择一些大厂头部的公司，这是用时间换钱效率最高的方式；如果学历和家庭背景都可以，可以选择国企（人际关系复杂）或外企（人际关系简单）；如果学历能力都不行，可以选择一些要求不高的初创公司作为跳板，努力学习工作，争取去更大的公司🧱。
</details>

<details>
  <summary>关于996</summary>

真996的公司并不多，比如某些公司可能是995或者大小周。如果薪资真的很高，那也还行，毕竟公司将你多余的时间买去同时给你更多的钱，这是一笔双方都愿意的交易。不过持续的996不只对身体不好，同时对脑子也不好，因为会耗尽你的心力，让你对很多事物失去了好奇心，久而久之，也在无形中错过了很多潜在的机会，这才是996最大的危害。等到你适应了这种工作节奏，公司会以某种理由优化掉你。所以当你身处这类公司时，时刻要考虑如何在这一天来临时做好准备。一方面当你996时，你没有更多的精力去发展新的技能，创造新的收入来源，你能做的就是利用好996高收入的优势做好积蓄和投资，让未来失业的日子更好过一些。

当然也有不少公司是996但是不给高工资，对于此类公司，要么尽快离开，要么掌握好摸鱼技能，只要摸鱼摸的好，你就能白嫖老板。
</details>

<details>
  <summary>工资的一些真相</summary>

工资究竟意味着什么？可能在当今的社会，很多人已经清楚的意识到工作并不会让社会变得更美好，可能会让老板活的更美好。工资的本质是：

- 资本买断你的部分时间，利用你的时间去创造更大的利润，然后给你定期发放利润的部分作为补偿。公司就是一个利用员工时间去创造金融杠杆的机器。
- 一份年薪20万的工作，本质上相当于一份本金200万年化10%投资收益的资金，你用自身的时间和技能作为代价拿到了这份200万的本金，风险点在于你的时间和技能会不断的受到通货膨胀的影响，最终在某个时间你失业了，再也拿不到这200万的本金。
- 理解了工作的本质后，工作这个游戏的玩法就是你需要在失业这个时间点来临的时候，能打造出另外一条可以获得等价工资收入的路子，比如副业、投资、创作收入与个人咨询等，这些都是你能够脱离公司这台机器独立去赚钱的方式。
</details>
